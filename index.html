Desafio DIO - HTML

O HTML HyperText Markup Language é a linguagem usada para estruturar o conteúdo de páginas na web. Uma de suas principais características é o uso de tags, que funcionam como instruções para o navegador sobre como apresentar textos, imagens, links e outros elementos. Recomendação: sempre consultar o www.w3.org/TR/html5.

O desafio consiste em:
 Criar título como linha separadora, como título e subtítulo com links, que seja redirecionado para outros na mesma página, acrescentar link de referencia. Entendendo o Desafio - A ideia deste Desafio de Projeto é elaborar um website, utilizando uma estrutura bem básica, utilizando tags aprendidas durante as aulas práticas: Utilizar todas as tags explicadas nas aulas: < h1> até < h6>, < p>, < mark>, < small>, < i>, < u>, < strong>, < ol>, < ul>, < li>, < a>, < hr>, < sub>, < sup>, < blockquote>; Utilizar novas tags sugeridas: < font>, < del>, < p>, < abbr> (a ideia é buscar estas tags na internet, entender como ela funciona e utilizar no texto).

Formatando textos

 voltar ao índice

As tags de título vão de < h1> até < h6> . A tag < h1> representa o título mais importante da página, geralmente usado para o cabeçalho principal, que nesse caso, foi utilizada na primeira linha, no título: Desafio DIO - HTML. As tags seguintes, < h2>, < h3>, < h4>, < h5> e < h6>, indicam subtítulos em ordem decrescente de importância, sendo < h6> o menos relevante. Essas tags ajudam a estruturar hierarquicamente o conteúdo, facilitando a leitura e a organização da informação. Nesse caso, foi utilizado < h3> nos subtítulos.

A tag < p> é utilizada para definir parágrafos. Ela separa blocos de texto e facilita a leitura, criando espaçamento automático entre os trechos.

Para destacar palavras ou trechos do texto, usamos algumas tags específicas. A tag < strong>, aplica ênfase forte, geralmente exibida em negrito, e tem significado semântico, indicando algo importante, como foi utilizado aqui no cabeçalho. Já a tag < i> representa texto em itálico, muitas vezes usado para palavras estrangeiras, nomes de obras ou termos que se deseja enfatizar de forma mais sutil, foi utilizado para marcar as tags propóstas nas instruções do desafio. A tag < mark> é usada para destacar visualmente partes do texto, geralmente com um fundo amarelo, como se fosse um marcador.

Quando queremos indicar siglas ou abreviações, usamos a tag < abbr >, que permite incluir o significado completo como dica ao passar o mouse sobre a palavra. Por exemplo, ao escrever “HTML”, pode-se usar HTML para tornar o texto mais informativo, permitindo deixar um texto maior 'escondido' que só é revelado quando se passa o mouse. É feito da seguinte forma: < abbr title="HyperText Markup Language">HTML< /abbr >, o texto desejado é escrito entre áspas duplas no 'title'.

Para elementos científicos ou matemáticos, como fórmulas químicas e expressões com expoentes, usamos as tags < sub> (para baixo) e < sup> (para cima). A tag < sub> coloca o texto em subscrito (abaixo da linha normal), como em “H2O” (H< sub>2< /sub>O), e a tag exibe texto em sobrescrito (acima da linha), como em “E = mc2” (E = mc< sup>2< /sup>).

As listas ordenadas, criadas com a tag < ol>, são usadas quando a ordem dos itens é importante, como em instruções passo a passo. Cada item da lista é marcado com < li>, formando uma sequência numerada. Já as listas não ordenadas, são marcadas com a tag < ul>, são usadas quando a ordem não importa, e os itens aparecem com marcadores (bolinhas ou outros símbolos). A lista criada acima está em formato < ul>, < li>.

Para inserir links, usamos a tag < a>, acompanhada do atributo href, ( < a hrsef = "" > < /a>) que define o endereço para onde o usuário será redirecionado ao clicar. Essa tag pode abrir outra página, ancorar para o assunto desejado dentro da mesma página, abrir um documento, entre outras funções dependendo do link.

A tag < hr> é usada para inserir uma linha horizontal,nfuncionando como uma separação visual entre seções do conteúdo como utilizado aqui na separação de assuntos nos subtítulos.

Por fim, a tag < blockquote> é usada para citações longas, geralmente exibidas com recuo no texto. Ela é apropriada quando se deseja destacar uma fala ou trecho de outro autor ou fonte.

Inserindo mídia

 voltar ao índice

O HTML permite inserir diversos tipos de mídia em uma página web, como imagens, áudios e vídeos.


TAG IMG

A tag < img > é usada para exibir imagens e precisa do atributo 'src' para indicar o caminho do arquivo.

Exemplo:

Inserimos imagens utilizando a tag 'img' e o atributo 'src' desta forma :
img src="imagem1desafio.jpg",
Os atributos 'width' e 'height', utilizamos para definir largura e altura da imagem
'title': aparece o título da imagem quando passa o mouse em cima, pode colocar oque quiser no título, nesse caso coloquei: title="Imagem de exemplo, possui três quadrados sendo um preto, um branco e um cinza".
'alt': descrição detalhada para leitor de tela (realizada por robô - Acessibilidade), com a seguinte mensagem: alt="Imagem de exemplo, imagem com filtros HTML + CSS, três quadrados, sendo o primeiro, imagem em preto, o segundo, imagem em branco, o terceiro, imagem em cinza".








TAG AUDIO

Para adicionar sons, usa-se a tag < audio>

Exemplo:
< audio controls>
< source src="Kiss - The Box Set - 15 - Nothing Can Keep Me From You.mp3"/>
Colocar essa mensagem para ser exibida se o navegador não tiver suporte: "Seu navegador não tem suporte para este áudio"
< /audio>
O atributo 'autoplay' ativa o áudio assim que a página é acessada: < audio autoplay controls >
Se tirar o 'controls' a barra de áudio some da página
Dá pra inserir o nome da música no 'title', e assim ela fica visível quando passar o mouse: title="KISS - Nothing can keep me from you"
Obs.: A aparência da barra de áudio varia de acordo com o navegador






TAG VIDEO



A tag < video > é utilizada de forma bem semelhante a tag de áudio.

Exemplo:
< video controls >
< source src="" 'widht' e 'height' para ajuste de largura e altura
Inserir essa mensagem para ser exibida se o navegador não tiver suporte: "Seu navegador não tem superte de vídeo"
Dá pra inserir o nome do vídeo no 'title' também, e assim ela fica visível quando passar o mouse: title=""
e para vídeos, a tag < video >, ambas com o atributo controls para mostrar os botões de play e pause. Essas mídias podem ser carregadas de arquivos locais ou de links externos.

Criando tabelas

 voltar ao índice

Nos anos 90, as tabelas eram utilizadas para estruturar o site, mas hoje não utilizamos mais pra essa finalidade, uma vez que estruturamos usando a semântica, CSS e JS. A tag 'table' sozinha não faz nada, apenas define o espaço que será tabelado. A tag 'tr' definirá as linhas, enquanto a tag 'td' definirá as colunas.


TABLE = tabela
A tag 'tr' = definirá as linhas
A tag 'td' = definirá as colunas

ATRIBUTOS:
'border' = cria borda
'width' = aumenta o tamanho da célula
'align' = centraliza a tabela (para centralizar o conteúdo da célula, o aling vai na frente do conteúdo)
'cellspacing' = espaçamento entre as células
'cellpadding' = espaçamento entre a borda e o conteúdo da célula
'sumary' = Nomeia o conteúdo sem exibir para o usuário
Tem a função de agrupar as partes da tabela:
TBODY = Corpo da tabela
THEAD = cabeçalho da tabela
TFOOT = rodapé da tabela
